# Basic Chatbot 

This is a simple rule-based chatbot built using Python.

# Objective
To create a chatbot that responds to user inputs using predefined rules.

# Features
- Responds to basic inputs like "hello", "how are you"
- Interactive conversation
- Exit command using "bye"

# Technologies Used
- Python
- If-Else Conditions
- Loops
- Functions

# How to Run
1. Run the Python file  
2. Type messages like "hello", "how are you"  
3. Type "bye" to exit  

# Example Output
# CodeAlpha_Basic_Chatbot
You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
